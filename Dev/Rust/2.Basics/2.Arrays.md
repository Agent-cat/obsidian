# arrays

- Creation of an array

```rust

let arr = [10, 20, 30, 40];

```

- we can also give it a type

```rust

let arr =[0;8];

```

- The above code means eight places with value 0

- `How to use ana array`

``` rust

fn main() {

let arr = [10, 20, 30, 40];

println!("{},{},{}", arr[1], arr[2], arr[3]);

}

  

```
